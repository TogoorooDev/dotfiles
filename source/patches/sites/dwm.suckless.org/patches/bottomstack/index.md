bottomstack
===========

Description
-----------
`bstack` and `bstackhoriz` are two stack layouts for dwm.

	bstack        (TTT)       bstackhoriz   (===)
	+-----------------+       +-----------------+
	|                 |       |                 |
	|                 |       |                 |
	|                 |       |                 |
	+-----+-----+-----+       +-----------------+
	|     |     |     |       +-----------------+
	|     |     |     |       +-----------------+
	+-----+-----+-----+       +-----------------+

`bstack` can be selected with [Alt]+[u], `bstackhoriz` with [Alt]+[o].
