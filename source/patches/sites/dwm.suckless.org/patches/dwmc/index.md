dwmc
====

Description
-----------
Simple dwmc client using a fork of
[fsignal](../fsignal/) to communicate with dwm.

Example
-------

See
[here](https://github.com/danielpza/dotfiles/blob/master/dwm/etc/xdg/sxhkd/sxhkdrc#L24)
for an example of usage with sxhkd.

	dwmc setmfact +0.05
