xrdb
====

Description
-----------
Allows dwm to read colors from `xrdb (.Xresources)` at run time.

X resources used:

	dwm.normbordercolor
	dwm.normbgcolor
	dwm.normfgcolor
	dwm.selbordercolor
	dwm.selbgcolor
	dwm.selfgcolor

Example Xresource string: `dwm.normbgcolor: #002b36`.

Default key to reload is `Mod+F5`.
