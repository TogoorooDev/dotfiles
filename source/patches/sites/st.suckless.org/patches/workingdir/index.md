workingdir
==========

Description
-----------
This patch allows user to specify the initial path st should use as WD
(working directory).

Examples
--------
Absolute path:
`st -d /usr/bin/`

Relative path:
`st -d dev/st/`
or
`st -d ~/dev/st/`
