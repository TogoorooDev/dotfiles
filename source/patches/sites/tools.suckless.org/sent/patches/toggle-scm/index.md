Toggle Colorscheme
==================

Description
-----------
This patch is meant to be applied on top of [inverted colors](https://tools.suckless.org/sent/patches/inverted-colors/).
It adds a new binding, "i", which lets you toggle between the two colorschemes
during presentation.
